# node-template
基于node+express 搭建的node服务平台、模板
### 项目技术栈： node+express
## package.json
```
{
    "name": "node-template",
    "version": "0.0.0",
    "private": true,
    "scripts": {
        "start": "nodemon ./bin/www"
    },
    "dependencies": {
        "cookie-parser": "~1.4.4",
        "debug": "~2.6.9",
        "express": "~4.16.1",
        "http-errors": "~1.6.3",
        "jade": "~1.11.0",
        "morgan": "~1.9.1"
    }
}

```
# 安装依赖
npm install

# 启动服务 localhost:3000(node默认端口是3000)
npm start
